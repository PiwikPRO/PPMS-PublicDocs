=====
Getting started
=====

Our JavaScript library has methods that let you use Consent Manager. You can use these methods in JavaScript.

In short, our JavaScript library lets you:

* Get compliance types
* Get and set complience settings
* Send a data request
* Open a consent form
* Track consent stats
