=================
startNewSession()
=================

The **startNewSession()** method starts a new session.

Syntax
------

.. tabs::

    .. group-tab:: Java

        .. code-block:: javascript

          getTracker().startNewSession();


    .. group-tab:: Kotlin

        .. code-block:: javascript

          tracker.startNewSession()
